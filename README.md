# O7 Suites CRM Landing Page

Premium landing page for O7 Suites CRM, built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Project structure

- `src/App.jsx`: landing page content and sections
- `src/App.css`: component-level styling
- `src/index.css`: global theme and resets
